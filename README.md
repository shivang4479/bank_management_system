"A Java-based Bank Management System simulating core ATM and retail banking functionalities with a GUI built using Swing 
and AWT, integrated with a MySQL database for persistent data storage."
Users can interact with the system to perform tasks such as:

Account Creation: A multi-page signup process capturing personal, additional, and account-specific details.
Authentication: A login system verifying card numbers and PINs.
Financial Operations: Depositing funds, withdrawing cash, and "Fast Cash" options.
Account Management: PIN changes, balance inquiries, and generating mini-statements.

Tech Stack: Java (Swing, AWT), MySQL.

Core Features:
User Registration: Multi-step signup for personal and account details.
Security: Secure login system verifying card numbers and PINs.
Transactions: Deposit, withdrawal, and "Fast Cash" options.
Account Management: PIN changes, balance inquiries, and mini-statements.
